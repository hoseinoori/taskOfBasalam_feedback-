# سیستم مدیریت فیدبک (Feedback Board) - تسک ورودی باسلام

این پروژه یک سیستم ساده و استاندارد برای ثبت، مشاهده و تایین وضیعیت فیدبک‌ها است. پروژه شامل دو بخش اصلی صفحه کاربر (ثبت فیدبک) و داشبورد مدیریت (تغییر وضعیت فیدبک‌ها) می‌باشد.

---

## تصمیم‌های فنی و دلیل آن‌ها

1. **بک‌آند (FastAPI):** به دلیل راحتی کار با ان و سرعت بالا و (Swagger UI)
2. **پایگاه داده (SQLite & SQLAlchemy):** از دیتابیس سبک SQLite به همراه ابزار SQLAlchemy استفاده شد. این انتخاب برای این بود که این پایگاه داده سبک تر است و برای نصب لوکال و تست ها بهتر است.
3. **فرانت‌آند (Single-File & Tailwind):** برای جلوگیری از پیچیدگی، ظاهر برنامه با HTML طراحی شد. استایل‌دهی به کمک کتابخانه Tailwind CSS انجام شده. کدهای جاوااسکریپت (Fetch API) به صورت درون‌برنامه‌ای (Inline) درون فایل‌های html نوشته شدند تا برنامه با یک دبل‌کلیک ساده آماده کار باشد.
4. **طراحی بر اساس اصول RESTful API**

---

## راهنمای اجرای پروژه در سیستم لوکال

برای اجرای پروژه روی سیستم خود، مراحل زیر را دنبال کنید:

### ۱. دانلود فایل ها

ابتدا پروژه را در سیستم خود دانلود کرده و وارد پوشه آن شوید:

</div>

```bash
git clone https://github.com/hoseinoori/taskOfBasalam_feedback-.git
cd taskOfBasalam_feedback-
```

<div dir="rtl">

### ۲. نصب پیش‌نیازها

پیش‌نیازهای پروژه را با اجرای کد زیر در ترمینال نصب کنید:

</div>

```bash
pip install fastapi uvicorn sqlalchemy
```

<div dir="rtl">

### ۳. اجرای سرور بک‌اند

بک‌آند پروژه را که در فایل `feedback.py` قرار دارد، با استفاده از ابزار Uvicorn روشن کنید:

</div>

```bash
uvicorn feedback:app --reload
```

<div dir="rtl">

### ۴. اجرای صفحات فرانت‌آند

با باز کردن فایل های فرانت در مرورگر کار تمام است.
